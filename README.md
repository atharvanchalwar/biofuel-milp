# Biofuel Logistics Supply Chain

The full report is present in the repository including sources and citations for all of the relevant databases used and methods followed.

Please refer to the Gurobi readme to find steps on how to run the MILP. Actual code used to scrape and collate the individual data files has not been provided in the repository however as mentioned above the sources are all available for reference in the report.
